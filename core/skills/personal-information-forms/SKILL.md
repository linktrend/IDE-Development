# Personal Information Forms

Use this skill whenever a form collects personal identity or contact data.

## Core Rules

1. Use stacked labels above controls.
2. Keep related fields grouped logically.
3. Show required state clearly.
4. Prefer inline validation over submit-only surprises.
5. Never rely on placeholder text alone to explain a field.

## Field Group Standards

### Name

- title
- first name
- middle name
- last name

### Address

- street line 1
- street line 2
- city
- state or region
- postal code
- country

### Phone

- country code
- phone number

## UX Standards

- keep forms concise
- ask for only necessary data
- place sensitive or uncommon fields later in the flow
- keep labels clear and readable
- make errors visible and accessible
